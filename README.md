# Motorola-Mobile-Sales-Power-BI-Dashboard
A Power BI Sales Analytics Dashboard built to analyze Motorola mobile sales performance across different Indian cities, time periods, customer ratings, and payment trends. The dashboard transforms raw sales data into meaningful insights that help identify best-performing regions, top mobile models, seasonal demand patterns, and customer purchase behavior.

# Project Overview
This project focuses on understanding sales performance trends for Motorola mobile phones. It provides a 360° view of sales metrics including:
1. Total Sales Revenue
2. Units Sold
3. Transaction Trends
4. City-wise Performance
5. Top Selling Models
6. Payment Method Insights
7. Customer Feedback Ratings
8. Monthly and Weekly Selling Patterns
The dashboard is fully interactive and designed to support business decision-making regarding marketing, inventory planning, and distribution strategies.

# Key Business Questions Answered

1. Which cities contribute the highest sales?
2. Which mobile models are the top performers?
3. Which payment methods are most preferred by customers?
4. How does sales quantity vary month-to-month?
5. What is the distribution of customer ratings?
6. What is the average transaction value?

# Tech Stack Used

1. Power Bi Desktop :- for Dashboard creation & data modeling
2. Power Query :- for Data cleaning & transformation
3. DAX (Data Analysis Expressions) :- for KPI calculations & custom measures

# Data Modeling Approach

1. Performed data cleaning using Power Query
2. Created relationships between fact and dimension tables
3. Used DAX measures for dynamic calculations like: Total Sales, Average Sales, Total Quantity, Customer Rating Distribution

# Key Insights Generated

1. Major sales concentrated in metro and Tier-1 cities like Mumbai, Delhi, Bangalore, Hyderabad.
2. iPhone SE, OnePlus Nord & Galaxy Note 20 emerged as top-selling models.
3. UPI and Debit Card are the most preferred payment methods (over 50% combined).
4. Higher customer satisfaction ratings indicate strong product acceptance.
5. Monday and Saturday show stronger sales activity — strategic days for promotions.

# Value Delivered

This dashboard helps stakeholders:

1. Optimize inventory distribution by city
2. Improve sales strategy and promotional timing
3. Align product pricing and customer satisfaction insights
4. Understand market demand behavior at a granular level
